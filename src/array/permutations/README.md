# Permutations of numbers

Given a collection of distinct numbers, return all possible permutations.

## Example,

[1,2,3] have the following permutations:

[			<br />

  [2,1,3],	<br />
  [2,3,1],	<br />
  [3,1,2],	<br />
  [3,2,1]	<br />
]
